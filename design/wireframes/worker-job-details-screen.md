## Key Elements

### Header
- Back button to return to search results
- Job title and wage information
- Employer profile/settings icons

### Job Information
- Clear display of job title and wage
- Visual representation (image/icon)
- Location and posting time
- Number of positions available

### Details Sections
- Description with key information
- Required skills in list format
- Work schedule and duration

### Employer Information
- Employer name and rating
- Direct contact options (call/message)

### Action Buttons
- Primary "APPLY FOR JOB" button
- Secondary "Save for Later" option

### Navigation
- Bottom navigation for easy access to main sections

## Accessibility Features
- Large, clear headings
- Sufficient contrast for text
- Large touch targets for buttons
- Simple, linear layout
- Clear visual hierarchy

## Interaction Flows
- Tapping "APPLY FOR JOB" starts application process
- Tapping "Call Employer" initiates phone call
- Tapping "Message Employer" opens chat
- Tapping "Save for Later" saves job to bookmarks
