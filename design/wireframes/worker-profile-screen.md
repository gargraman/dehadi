## Key Elements

### Header
- Profile title
- Message and settings icons

### Personal Information
- Large profile picture
- Name, phone number, and location
- Clear, editable fields

### Earnings Summary
- Visual blocks showing completed jobs and total earnings
- Simple, digestible information

### Skills Section
- List of worker's skills
- Edit button for skill management

### Recent Activity
- Timeline of recent job activities
- View all button for complete history

### Quick Actions
- Important account management options
- Language and notification settings

### Navigation
- Bottom navigation (current page highlighted)

## Accessibility Features
- Large profile picture
- Clear text hierarchy
- Sufficient spacing between sections
- Large buttons for actions
- Simple, scannable layout

## Interaction Flows
- Tapping "Edit Skills" opens skill management
- Tapping "Edit Profile" opens profile editor
- Tapping "Payment Info" opens payment details
- Tapping "View All Activity" shows full history
